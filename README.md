## Manual Installation
```
git clone git@github.com:jamjnsn/dotfiles.git .dotfiles
cd .dotfiles
chmod +x install.sh
./install.sh
```

## Automatic Installation
```
curl https://raw.githubusercontent.com/jamjnsn/dotfiles/main/install.sh | bash
```
